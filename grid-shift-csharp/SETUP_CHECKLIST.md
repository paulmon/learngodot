# Setup Checklist (Godot C#)

- [ ] Install Godot 4.x .NET build
- [ ] Ensure `dotnet --version` works
- [ ] Add Godot executable to PATH (optional but recommended)
- [ ] Create/open Godot project in this folder
- [ ] Verify one C# script compiles
- [ ] Configure Input Map actions
- [ ] Create `Main.tscn` with Node2D root
- [ ] Add placeholder TileMapLayer and player node
- [ ] Run project successfully

## Troubleshooting
- If Godot command is not found in terminal, launch Godot via UI and continue setup.
- If C# build fails in Godot, verify your installed .NET SDK matches Godot requirements.
